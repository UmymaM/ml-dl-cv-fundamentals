# ml-dl-cv-fundamentals
This repo covers the foundational topics in machine learning, deep learning and computer vision through hands-on projects

It contains multiple projects organized by domain:

- **Phase 1 — ML Fundamentals:** Regression, classification, NLP, tabular data pipelines.
- **Phase 2 — Computer Vision Basics:** Image processing, face detection, landmark extraction, augmentation.
- **Phase 3 — Deep Learning & CNNs:** Custom CNNs, transfer learning, Siamese networks, autoencoders, emotion recognition.
- **Phase 4 — Generative Models & Advanced Vision:** GANs, face swapping, temporal analysis, vision transformers.

## Tools & Libraries
Python, PyTorch, scikit-learn, OpenCV, dlib, MediaPipe, Albumentations, torchvision, matplotlib, pandas, NumPy
